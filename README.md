# Property-Analysis

Financial modeling tools for short-term rental property evaluation. Built for west Michigan markets (Whitehall, St. Joseph).

## Tools

### [STR Financial Model](https://moviebuff24.github.io/Property-Analysis/str-financial-model.html)
Interactive analyzer with revenue scenarios, occupancy sensitivity, and break-even analysis. Enter a property address, purchase price, and key assumptions to model cash flow across base, optimistic, and conservative cases.
